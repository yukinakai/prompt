# プロンプト集リポジトリ

このリポジトリは、日頃AIで利用しているプロンプトをまとめたコレクションです。試行錯誤で得たプロンプトを整理し、再利用や共有をしやすくすることを目的としています。

- `chatgpt/` : ChatGPT関連のプロンプトや設定を置く場所
- `notebook_lm/` : ノートブックやLLM検証用のプロンプト置き場

## 使い方

1. 目的に合うディレクトリを開き、プロンプトファイルをコピーして利用してください。
2. 使ってみて改善点があれば、同じ場所に追記や新規ファイルとして追加してください。
3. 変更を共有したい場合は、変更内容と意図を簡潔に書き添えてコミットすると管理がしやすくなります。

## 運用メモ

- 新しく試したプロンプトは失敗例も含めて残しておくと、後で比較検証がしやすくなります。
- 似た用途のプロンプトが増えてきたら、READMEに簡単な一覧を追記して整理してください。
